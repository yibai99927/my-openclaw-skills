# Finance News (安全版)

市场新闻获取工具，仅使用公开API，无敏感配置。

## 功能

- 📰 市场概况 - 显示主要指数实时行情
- 📊 股票新闻 - 获取特定股票新闻
- 📡 新闻源 - 查看可用新闻源

## 使用方法

```bash
# 市场概况
finance-news-safe market

# 股票新闻
finance-news-safe news AAPL

# 查看新闻源
finance-news-safe sources
```

## 安全说明

- ✅ 无需API Key (演示模式)
- ✅ 无敏感配置
- ✅ 无消息投递功能
- ✅ 仅网络查询

## 数据来源

- Yahoo Finance
- Finnhub (演示模式)
